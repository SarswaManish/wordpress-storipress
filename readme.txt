Export your WordPress data to Storipress by one-click.

== Installation ==

1. Use Add New Plugin in the WordPress admin panel
2. Activate the plugin through "Plugins" menu in WordPress
3. Click `Tools -> Export to Storipress`

== Frequently Asked Questions ==

= How do I use this? =

After installing and activating this plugin, go to Tools -> Export to Storipress.
